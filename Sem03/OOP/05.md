# Function calls

## Call by value
``` fun(int x)```\n
Passing the value of variable

## Call by Address
``` fun(int *x)```
Passing the address of variable

## Call by reference
``` fun(int &x)```
Passing the variable itself
Essentially saying ``` x = x``` (in main and fun)
